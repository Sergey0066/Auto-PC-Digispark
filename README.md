# Auto-PC-Digispark
На основе платы Digispark сделаем автозапуск ПК при подаче электричества! Когда на ПК снова поступил электричество, он автоматически запустится без вашего участия! Полезно для домашних серверов, у которых постоянно перебои электричества)

# Начало
Для начала нужно купить необходимые запчасти для нашего устройства.
- 1. Плата Digispark USB (1 штука)

- 2. Оптопара PC817 (1 штука)

- 3. Резистор 2 вт 390 Ом (1 штука)

- 4. Комплект соединительных проводов Dupont
 
![photo_2026-01-31_16-24-06](https://github.com/user-attachments/assets/983ccbef-500b-48cd-95b3-32eab6ece943)


- 5. Печатная макетная плата (1 штука)

- 6. Набор для пайки с паяльником.
 
# Схема

   5V источник
   +5V   GND
    |     |
    |     |
   5V    GND
    |     |
 +--+-----+--+
 |  Digispark |
 |            |
 | P0--[390]--+--> PC817 pin 1
 |            |
 | GND--------+--> PC817 pin 2
 +------------+

            PC817
         +----------+
         |          |
         | 4 ----+ |----> PWR_SW
         |        | |
         | 3 ----+ |----> PWR_SW
         |          |
         +----------+

# КОД для Digispark
Делать через Visual Studio Code с расширением platform io.

Заменить содержимое /src/main.cpp на код ниже:

void setup() {
  pinMode(0, OUTPUT);
  digitalWrite(0, LOW);
  delay(1500);
  digitalWrite(0, HIGH);
  delay(200);
  digitalWrite(0, LOW);
}
void loop() {}
