# Auto-PC-Digispark
На основе платы Digispark сделаем автозапуск ПК при подаче электричества! Когда на ПК снова поступил электричество, он автоматически запустится без вашего участия! Полезно для домашних серверов, у которых постоянно перебои электричества)

# Начало
Для начала нужно купить необходимые запчасти для нашего устройства.
- Плата Digispark USB (1 штука)

- Оптопара PC817 (1 штука)

- Резистор 2 вт 390 Ом (1 штука)

- Комплект соединительных проводов Dupont

- Печатная макетная плата (1 штука)

- Набор для пайки с паяльником.
 
# Схема
```
  Воткнуть в usb
    |     |
    |     |
    |     |
    |     |
 +--+-----+--+
 |  Digispark |
 |            |
 | P0--[390]--+--> PC817 pin 1
 |            |
 | GND--------+--> PC817 pin 2
 +------------+

            PC817
         +----------+
         |          |
         | 4 ----+ |----> PWR_SW
         |        | |
         | 3 ----+ |----> PWR_SW
         |          |
         +----------+
```

# Код для Digispark
Делать через Visual Studio Code с расширением platform io.

Заменить содержимое /src/main.cpp на код ниже:

```
void setup() {
  pinMode(0, OUTPUT);
  digitalWrite(0, LOW);
  delay(1500);
  digitalWrite(0, HIGH);
  delay(200);
  digitalWrite(0, LOW);
}
void loop() {}
```
